The Challenge

AI agents can reason.
They cannot remember.

The problem is not intelligence. The problem is coherence. Every session starts from zero — no persistent identity, no verifiable record, no structured understanding of how knowledge relates across time.

01
No Memory

Each session is stateless. Knowledge from one conversation is unavailable in the next. RAG retrieves blobs — it cannot reason over structure.

02
No Identity

Who created this? When? Has it been modified? There is no provenance, no authorship, no tamper-evidence baked into the data layer.

03
No Reasoning Substrate

Documents live as blobs. There is no representation of causal chains, temporal sequences, contradictions, or community structure.

AION solves this with five primitives

File Link Context Journal Snapshot

Core Primitives

Five primitives.
Infinite composition.

Every AION capability — from a collaborative wiki to a forensic AML investigation — is built from the same five primitives. Complexity emerges from their composition.

01
File

An annotated, content-addressed blob backed by a .aion capsule. Any bytes: text, image, code. Signed at creation. Identified by its BLAKE3 hash — the content is the address.

02
Link

A directed, typed edge in the knowledge graph. Four kinds: causal, temporal, semantic, provenance. Links give nodes meaning in relation to other nodes.

03
Context

A named lens over the graph — a curated subset of nodes. Contexts scope investigations, retrievals, and domains. Denied nodes are filtered automatically.

04
Journal

An ordered temporal sequence of events — an episode. Journals capture what happened, when, and in what order. Append-only; entries are immutable.

05
Snapshot

A signed truth checkpoint. BLAKE3 hash of the entire graph state, signed with Ed25519, inserted as a provenance node. Tamper-evident chain of custody.

The Storage Layer

Cryptographic identity
baked in at rest.

Cryptographic Primitives

Ed25519 Digital Signatures Every capsule is signed. Verification is constant-time — no early-exit on byte comparison. Keys are zeroized on drop.
BLAKE3 Content Addressing The NodeId is the BLAKE3 hash of the payload. Identical content = identical id. Snapshot digests chain BLAKE3 over full graph state.
XChaCha20‑Poly1305 Authenticated Encryption At-rest encryption. 192-bit nonces, randomly generated per encryption, never reused. AEAD: tampering detected on decryption.
ML-DSA Post-Quantum Signing CRYSTALS-Dilithium. NIST-standardized. Available as an upgrade path when quantum resistance is required.

Non-negotiable security rules

No cryptographic primitive implemented from scratch — all from the RustCrypto ecosystem. All key material is zeroized on drop. Test vectors from RFC/NIST are included for every function.

MCP Interface

Eighteen tools.
One coherent interface.

Every primitive, every analytic, every policy action is a typed MCP call. Agents compose these tools the same way a developer composes functions.

Graph Topology & Traversal

aion_linkCreate a typed directed edge — causal / temporal / semantic / provenanceGraph
aion_pathShortest path between two nodes; returns each hop and edge kindGraph
aion_impactForward or backward blast radius from a node across N hopsGraph
aion_communityLabel-propagation community detection — cluster membership for all nodesGraph
aion_contradictionsDetect temporal cycles — edges forming impossible chronological loopsGraph

aion_recallSpreading activation from seed nodes — surfaces structurally related nodesIntel

Temporal · Policy · Audit
